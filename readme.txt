== Description ==

The **"Search Engines Blocked in Header"** plugin enhances your WordPress experience by displaying the "Search Engines Discouraged" notification directly in the WordPress Toolbar. This notification, previously located in the Right Now box starting from **WordPress 3.2**, ensures that you are always aware of your site's search engine visibility status.

= Key Features =

* **Convenient Notification**: Keep track of your site's search engine visibility status directly in the WordPress Toolbar.
* **Seamless Integration**: The plugin seamlessly integrates with WordPress, providing a hassle-free experience.
* **Easy Accessibility**: Instantly access the "Search Engines Discouraged" notification without navigating through multiple screens.
* **Translation Support**: Easily translates the notification message to your preferred language to match your site's language.

By simply activating this plugin, you'll have immediate access to vital information about whether search engines are discouraged from indexing your site. This is particularly useful for site administrators and super admins who need to monitor the visibility settings of their WordPress sites closely.

== Installation ==

1. Upload `search-engines-blocked-in-header` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. We're all done.

== Frequently Asked Questions ==

= What does the "Search Engines Discouraged" notification indicate? =

The "Search Engines Discouraged" notification indicates that search engines are currently discouraged from indexing your WordPress site. This setting can impact your site's visibility in search engine results.
